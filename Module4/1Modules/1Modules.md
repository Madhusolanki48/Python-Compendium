###  **Modules in Python**

- **Definition**:  
  A **module** is a Python file (`.py`) containing definitions (functions, variables, classes) and statements that can be reused in other programs.

- **Purpose of Modules**:  
  - Add extra functionality to Python programs  
  - Promote **modular programming** (breaking code into manageable parts)

- **Advantages**:
  -  **Scope**: Each module has its own namespace  
  -  **Reusability**: Avoids code duplication, improves efficiency  
  -  **Simplicity**: Modules have clear, specific purposes and low interdependency

- **Types of Modules**:
  - **Built-in modules** (e.g., `math`, `re`, `os`)  
  - **User-defined modules** (custom `.py` files)  
  - **External modules** (e.g., `matplotlib`, installed via pip)

- **Usage**:
  - Import using `import module_name`  
  - Modules are imported only **once** during execution  
  - Must import before calling any function from the module  
  - Can be imported globally or inside functions

- **Examples**:
  - `import math` → use `math.gcd()`, `math.factorial()`  
  - `import re` → for regular expression operations  
  - `import matplotlib.pyplot as plt` → for data visualization
